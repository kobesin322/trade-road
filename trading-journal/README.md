# Trading Journal Dashboard

A gamified March 2026 trading journal built with Next.js 15, TypeScript, Tailwind CSS, shadcn-style UI primitives, Recharts, Lucide icons, and date-fns.

## Features

- Dark, neon dashboard with total profit, win rate, trade count, and streak cards.
- Strategy and daily profit charts powered by Recharts.
- Searchable, filterable trade journal with win/loss and strategy tabs.
- Trade chart overview with mini market-path cards.
- March 2026 calendar view with per-day WIN/LOSS return badges.
- Local storage persistence seeded with 23 realistic sample trades.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
